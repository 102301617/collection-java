#### Java数据分为两大类

1. 基本数据

2. 引用数据

###### 基本数据

整数

```java
int num1 =10; //最常用  
byte num2 =20; //范围小：-128~127
short num3 =30;
long num4 =30L; //long类型数字后面加L
```

小数/浮点数

```java
float num5 =50.1F; //后面加F
double num6 =50.2;
```

字符

```java
char name ="a";
```

字符串

```java
String name ="字符串";
```

布尔值：代表是非

```java
boolean flag = true;
boolean flag = false;
```

进制：

二进制0b开头 十进制 八进制0开头 十六进制0x开头



浮点是有限 离散 舍入误差 大约 接近但不等于的

所以 <mark>最好不用浮点数进行比较



常量

```java
final =1; //常量只能赋值一次 固定不变
```
